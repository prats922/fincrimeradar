# FinCrimeRadar 🛡️

**Free, open-source AML and sanctions screening for everyone.**

FinCrimeRadar is a free educational tool for compliance students and professionals who want to understand how real-world financial crime screening works — sanctions checks, PEP screening, and adverse media monitoring, all powered by live public data.

🌐 **Live at:** [fincrimeradar.org](https://fincrimeradar.org)

## Data sources

FinCrimeRadar uses verified, publicly available datasets:

- **OFAC SDN** — US Treasury sanctions list
- **UN Consolidated List** — UN Security Council sanctions
- **EU Financial Sanctions** — European Union list
- **OFSI** — UK Office of Financial Sanctions Implementation
- **OpenSanctions** — Open-source global sanctions aggregator
- **Wikidata** — PEP data
- **GDELT** — Global news and adverse media
- **World Bank Debarred** — Debarred entities list

## Tech stack (planned)

- **Backend:** Python + FastAPI
- **Screening engine:** OpenSanctions API + fuzzy matching (rapidfuzz)
- **Frontend:** HTML/CSS/JS (no framework, keep it simple)
- **Hosting:** Vercel (frontend) + Render (API)
- **Database:** PostgreSQL
